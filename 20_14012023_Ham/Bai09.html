Bài 9: Viết hàm truyền vào mảng số nguyên bất kỳ, 
        sau đó đảo ngược mảng số nguyên đó.
        VD: [1, 9, -3, 2, 0, 4, 11]
        ==> [11, 4, 0, 2, -3. 9, 1]
        [0] <===> [6 - 0]       ==> [11, 9, -3, 2, 0, 4, 0]
        [1] <===> [6 - 1]       ==> [11, 4, -3, 2, 0, 9, 0]
        [2] <===> [6 - 2] = [4] ==> [11, 4, 0, 2, -3, 9, 0]
        [3] <===> [6 - 3] =[3]  ==> [11, 4, 0, 2, -3, 9, 0]
